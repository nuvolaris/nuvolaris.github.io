Local Installation

This section describes how to install Nuvolaris on a remote server

Prerequisite

Before installing, you need to:

  • generate an ssh key to access a server

  • provision a server adding the ssh key to access it

  • configure the services you want to enable

  • install nuv.

Installation

nuv setup server <host> [<user>]

Update

After installation you can update adding more services== Uninstall

Uninstall

nuv setup server <host> [<user>] --uninstall
