Nuvolaris

Welcome to Nuvolaris!

Nuvolaris is an Open Source project (Apache 2.0) that lets you easily build a complete and portable Serverless environment that runs in every Kubernetes.

Out of the box Nuvolaris contains the Apache OpenWhisk platform as its serverless engine, and on top you can add many optional services such as databases, message brokers, a cron scheduler and multi-tenancy. The platform is paired with an incredibly powerful CLI tool that can deploy Nuvolaris everywhere with one line and manage it easily.

We are building a complete distribution of a serverless environment that:

  • It is easy to install and manage.

  • Potentially runs in every Kubernetes.

  • It is tested regularly against a set of supported Kubernetes environments.

  • Includes a (possibly large) number of integrated services.

This is the main difference from Apache OpenWhisk, as it only provides a basic engine for serverless. If you want to know more about our goals are described in our roadmap document to read about.
