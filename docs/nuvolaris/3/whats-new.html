What’s New in Nuvolaris 3

  • A rich operator, supporting the deployment of integrated services:

    • Redis, a storage service

    • MinIO, an object storage service

    • Postgres a relational database

    • FerretDB A MongoDB compatible adapter

  • A completely rewritten, rich and extendible CLI, nuv. It currently supports Nuvolaris installation on:

    • Docker(/), using Kind

    • Amazon EKS

    • Azure AKS

    • Google GKE

    • RedHat OpenShift

    • Canonical MickoK8S

    • Rancher (now SuSE) K3S
