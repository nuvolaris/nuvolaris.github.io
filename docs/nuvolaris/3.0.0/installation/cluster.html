Cluster Installation

This section describes how to install Nuvolaris on a Kubernetes Cluster

Prerequisite

Before installing, you need to:

  • configure the services you want to enable

  • install nuv.

  • provision a Kubernetes cluster server retrieving its kubeconfig

Installation

nuv setup cluster [<context>]

Update

After installation you can update adding more services.

Uninstall

nuv setup cluster --uninstall
